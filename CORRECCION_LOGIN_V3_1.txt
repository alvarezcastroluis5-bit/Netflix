CORRECCIÓN V3.1 - BOTÓN INGRESAR

PROBLEMA CORREGIDO
La versión anterior cargaba Supabase mediante una importación ESM externa.
Cuando esa importación no se iniciaba, los botones Ingresar de ambos paneles
quedaban sin funcionamiento.

ARCHIVOS QUE DEBES REEMPLAZAR EN GITHUB
1. admin.html
2. revendedores.html
3. js/core.js
4. js/admin.js
5. js/reseller.js

También puedes reemplazar todo usando el ZIP completo.

DESPUÉS
1. Espera que GitHub Pages termine el despliegue.
2. Abre admin.html o revendedores.html.
3. Presiona Ctrl + F5.
4. Prueba nuevamente.

No necesitas ejecutar otro bloque SQL.
No necesitas volver a modificar hyper-processor para esta corrección.
